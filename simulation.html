Key Finding: Bracket Game Timing

From GSL Tournaments: "If the previous game ends early, tournament staff will let you know the adjusted start time which will never be more than 15 minutes prior to the scheduled game time."

This suggests bracket games use fixed scheduled times with minor adjustments (max 15 min early).

Simulation Assumptions

These assumptions are based on research but may not reflect all tournament variations.

  • Tournament format: 16-team tournament with 4 pools of 4 teams, top 2 from each pool advance to 8-team single elimination bracket
  • Pool play: Each team plays 3 games (round robin within pool). Schedule is fixed in advance.
  • Bracket play: Games have fixed scheduled start times. If previous game runs early, next game may start up to 15 min early.
  • Game duration variance: Games run -10 to +15 minutes from expected duration (slightly skewed toward running long)
  • Buffer between games: 20 minutes scheduled between games on same field
  • Booking window: Team books excursion 2+ hours after their last known game ends
  • Conflict definition: Booking overlaps with actual game time (including 30 min pre-game arrival)

Key Insights

Methodology

How the Simulation Works

  1. Generate Tournament: Create pools, assign teams, build bracket structure
  2. Schedule Pool Play: Assign fixed game times across available fields (Day 1)
  3. Simulate Games: Random win/lose outcomes, random duration variance (-10 to +15 min)
  4. Build Bracket: Top teams from each pool advance, seeded by record
  5. Schedule Bracket: Assign fixed times, but actual start may shift based on previous game
  6. Simulate Bookings: Each advancing team "books" an excursion 2 hours after their expected free time
  7. Detect Conflicts: Check if booking time overlaps with actual next game time (including 30 min arrival buffer)
  8. Aggregate Results: Run N simulations, calculate conflict rates by phase and strategy

Limitations

  • Real tournaments have more complex scheduling constraints (referee availability, team requests, etc.)
  • Game duration variance may differ by sport, weather, and competitiveness of match
  • Some tournaments use rolling start times, others use strict fixed times
  • Doesn't account for consolation brackets or 3rd place games
  • Booking behavior assumptions are estimates, not based on real user data
