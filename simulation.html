RoadGame Format Rules

These rules are what make 0% conflicts possible. They're not suggestions—they're the specification that defines the format.

  • Rule 1: Protected Activity Windows Specific time blocks are designated as "Activity Windows." No games can be scheduled during these times. Period.
  • Rule 2: Games Never Start Early The scheduled start time is the earliest a game can begin. If previous games finish early, the field sits empty until the scheduled time.
  • Rule 3: Minimum 2-Hour Gaps Between Bracket Rounds When a team advances, they're guaranteed at least 2 hours before their next game—enough time for a real activity.
  • Rule 4: 3-Hour Morning Activity Block Every day starts with a 3-hour protected window (8-11 AM) for team activities. No exceptions.
  • Rule 5: Published Schedule is Final Activity windows are published before the tournament. Families can book activities the moment registration opens.

The RoadGame Guarantee

Book activities during any protected window. Zero conflicts. 100% confidence. By design.
